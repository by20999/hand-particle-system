# Hand Particle System 更新文档

## 2026-05-22 性能、结构与移动端体验优化

### 本次更新概览

- 新增自动画质策略：桌面端默认约 `76k` 粒子，移动端约 `44k` 粒子，高 DPR 中等设备约 `62k` 粒子。
- 支持通过 URL 强制指定画质：`?quality=compact`、`?quality=balanced`、`?quality=desktop`、`?quality=ultra`。
- 优化粒子更新逻辑，预计算随机权重、目标方向、扩散参数，减少每帧重复计算。
- 移动端控制面板默认折叠，保留状态和手势条，让主体粒子画面更完整。
- 拆分 `src/main.js`，新增配置、粒子、模型、手势、光照和 UI 模块，降低维护成本。
- MediaPipe 改为动态导入，并在构建时拆分为独立 chunk。
- Vite 构建拆分 `three` 和 `hands` chunk，改善缓存和构建输出。
- 升级 `npm run verify`：可自动启动开发服务器、生成桌面/移动端截图、检查 WebGL 画面非空、捕获控制台错误和 404。
- 新增 `npm run check`，用于一键执行构建和截图验证。
- 调整脚本：`npm run dev` 默认本机开发，`npm run dev:host` 用于局域网访问。
- README 补充画质策略、隐私说明、脚本说明和新项目结构。

### 新增文件

- `src/config.js`：画质档位、模型缩放、MediaPipe 资源路径。
- `src/particles.js`：粒子缓冲区、Shader、预计算数据和粒子动画更新。
- `src/shapes.js`：爱心、花朵、土星、佛像、烟花等目标点生成。
- `src/gestures.js`：手势张合、掌心位置、拳头视角控制辅助函数。
- `src/lighting.js`：背景柔光、光源颜色、呼吸动画。
- `src/ui.js`：面板缩放、移动端折叠、状态文字和手势百分比。
- `vite.config.js`：Vite 服务和构建分包配置。

### 修改文件

- `src/main.js`：改为主循环、场景组装、摄像头初始化和事件绑定入口。
- `index.html`：增加面板折叠按钮、画质显示和新的面板结构。
- `src/styles.css`：增加移动端折叠面板、画质行和按钮布局样式。
- `scripts/verify.mjs`：改造成自动化验证脚本。
- `package.json`：新增 `dev:host`、`preview:host`、`check`，整理依赖位置。
- `package-lock.json`：同步依赖分类变化。
- `README.md`：更新运行说明、画质策略、项目结构和开发提示。
