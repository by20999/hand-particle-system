## 2026-05-28 GLB/glTF 贴图感知采样

### 本次更新概览

- GLB/glTF 点云采样从“网格形状 + 基础材质色”升级为“网格形状 + UV 贴图颜色”。
- 采样三角面时会记录材质、材质组、顶点色和 UV；每个粒子点用三角形重心坐标插值 UV，再从 base color 贴图读取对应像素颜色。
- 支持多材质 mesh 的 `geometry.groups`，不同三角面会使用对应材质和贴图，不再只取第一个材质颜色。
- 支持透明贴图和 `alphaMap`：透明区域会降低粒子透明度，并在采样时多次避开明显透明区域，减少无效粒子。
- 支持 `emissiveMap` 亮度加成，让自发光纹理在粒子点云里更亮一些。
- 导入完成提示会标明“已还原贴图颜色”，方便判断当前模型是否成功读取到贴图。

### 修改文件

- `src/main.js`：新增 GLB/glTF 材质组解析、贴图像素读取、UV 插值采样、透明贴图处理和 emissive 亮度加成。
- `README.md` / `AGENTS.md` / `更新文档.md`：同步贴图感知 GLB/glTF 采样说明。

## 2026-05-28 食指指向模式增强

### 本次更新概览

- 新增独立“指向模式”：识别到食指指向后，模型会快速跟随指向方向移动。
- 指向模式会临时压低张开扩散、禁用拳头视角控制，并降低 shader 内部指针扰动，避免模型一边移动一边被打散。

- 交互面板新增“指向灵敏度”滑条，单独调节指向移动幅度，不影响张开/收拢扩散灵敏度。
